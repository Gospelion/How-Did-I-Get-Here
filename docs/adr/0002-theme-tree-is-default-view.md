# 0002. Theme Tree Is The Default View

## Status

Accepted

## Context

A full node graph can communicate relationships, but it becomes noisy quickly when a research session includes many pages. The MVP needs to explain research structure at a glance.

## Decision

The side panel defaults to a Research Topic list and expandable topic tree. Page-to-page relationships are shown inside page details instead of as the primary view.

## Consequences

The interface is less visually dramatic than a graph, but it remains useful with dozens of pages and can still add a graph view later.
