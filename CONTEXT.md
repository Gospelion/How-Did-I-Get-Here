# Context

## Glossary

### Research Topic
A group of pages that belong to the same research task.

### Page Node
A single recorded web page and its reading state.

### Page Edge
A relationship between two Page Nodes, such as source navigation or shared topic.

### Core Page
A Page Node that is important enough to reopen first when resuming a Research Topic.

### Todo Page
A Page Node that the user may still need to read or revisit.
